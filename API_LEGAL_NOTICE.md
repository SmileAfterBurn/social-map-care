## 2. Ліцензійний режим (Public License UA)

Використання API регулюється **Публічною ліцензією UA**, яка надає право на некомерційне використання за таких умов:

### 2.1. Дозволені дії:
- Використання API для надання безкоштовної допомоги ВПО та вразливим групам.
- Інтеграція функцій пошуку соціальних послуг у волонтерські додатки.
- Посилання на API як на джерело верифікованих даних.

### 2.2. Заборони:
- **Комерціалізація:** Заборонено використовувати API у платних продуктах або для отримання рекламного доходу.
- **Реверс-інжиніринг:** Заборонено копіювати системні інструкції (System Instructions) моделі пані Думки для створення аналогічних сервісів.
- **Експорт бази:** Заборонено масовий автоматизований збір (scraping) даних через API для створення сторонніх баз даних.

---

## 3. Захист алгоритмів та ШІ-логіки

Технічний дизайн API включає унікальні методи трансформації даних, що є комерційною таємницею та інтелектуальним здобутком власника:
1. **Емоційний праймінг:** Специфічні алгоритми налаштування TTS для відтворення українського жіночого голосу.
2. **Логіка заземлення (Grounding Logic):** Алгоритм вибору між моделями Flash та Pro залежно від критичності запиту.
3. **Система рефералів:** Унікальний метод генерації анонімних кодів справ для захисту персональних даних клієнтів.
